개요

node.js를 설치할 때 nvm을 이용하여 설치 한다.

이유

node.js의 LTS버전은 그 기한이 짧다.
따라서 작년에 개발한 node.js소스와 올해 개발할 node.js의 버전이 달라지는 경우가 있는데 이미 설치된 node.js 버전을 변경 하면서 개발할 방법이 없다.
이를 nvm을 이용하여 node.js를 설치하게 되면 간단히 node.js 버전을 변경하면서 개발이 가능하다.

설치방법

nvm은 리눅스에서도 설치가 가능하고 윈도우에서도 설치가 가능하다.
아래 내용은 개발용 윈도우 PC에서 설치하는 것으로 간주 한다.

nvm 설치

  1. 사이트 접속: https://github.com/coreybutler/nvm-windows/releases
  2. 설치파일 다운로드: nvm-setup.zip
  3. 다운받은 파일 압축 해제 후 설치: nvm-setup.exe

nvm을 통한 node.js 설치

설치전에 node.js의 LTS 버전을 미리 확인할 것

> nvm install 14.15.4 64

(node.js 14.15.4 버전을 64비트용으로 설치한다는 뜻)

nvm을 통해 설치된 node.js 버전 선택

> nvm use 14.15.4

nvm을 통해 설치된 node.js 리스트 확인

> nvm list
  * 14.15.4 (Currently using 64-bit executable)
    12.20.0

위와 같이 현재 사용되는 버전 및 리스트를 확인할 수 있다.
